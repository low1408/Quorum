# Extra

This folder contains sanitized supplemental material for project submission.

## AI Chat Session Logs

Shareable AI council reports are stored in `ai-chat-logs/`. These files are copied from generated Markdown reports after review.

Do not include:

- encrypted session files from `sessions/`
- real `.env` files
- local databases such as `orchestrator.db`
- private keys or access tokens
- raw logs that may contain secrets

If assessors require raw AI chat history, export it as sanitized Markdown before adding it here.
